## Permohonan Semakan kepada Pengurus Program
Peranan: Penyedia Program

Pra-syarat:
* Semua aktiviti di bawah program telah diluluskan (Rujuk kepada **Status Semakan** pada sudut kanan atas halaman program)
* Maklumat latar belakang dan outcome telah memenuhi syarat-syarat yang ditetapkan

Langkah:
1. Klik **Program** pada *Sidebar*
2. Pilih program yang berkaitan
3. Klik **Kerangka Prestasi** pada sudut kanan atas halaman program
4. Klik butang **Semakan**  
5. Klik butang **Maklum semak**
6. Pastikan mesej **Berjaya** dipaparkan di bawah halaman

Outcome:
* Pengurus Program akan menerima notifikasi dalam sistem untuk membuat semakan 

## Semakan Pengurus Program
Peranan: Pengurus Program

Pra-syarat:
* Semua aktiviti di bawah program telah diluluskan (Merujuk kepada **Status Semakan** pada sudut kanan atas halaman program)
* Maklumat latar belakang dan outcome telah memenuhi syarat-syarat yang ditetapkan

Langkah:
1. Klik **Program** pada *Sidebar*
2. Pilih program yang berkaitan
3. Klik **Kerangka Prestasi** pada sudut kanan atas halaman program
4. Klik butang **Semakan** 
5. Kunci masuk **Keterangan** dan buat pilihan untuk **Luluskan/Mohon Pindaan**
6. Klik butang **Kemaskini**
7. Klik butang **Luluskan** sekiranya **Luluskan** dipilih / butang **Teruskan** sekiranya **Mohon pindaan** dipilih
8. Pastikan status semakan telah dikemaskini kepada **Luluskan(Hijau)/Perlu pindaan(Oren)**

Outcome:
* Penyedia Program akan menerima notifikasi sekiranya pindaan diperlu
* Penyedia Program, Penyedia MES dan Pegawai Pengawal akan menerima notifikasi sekiranya kelulusan diberikan


## Timpa Keputusan Pengurus Program
Peranan: Penyedia MES / Pegawai Pengawal

Pra-syarat:
* Program yang berkaitan telah diluluskan

Langkah:
1. Klik **Program** pada *Sidebar*
2. Pilih program yang berkaitan
3. Klik **Kerangka Prestasi** pada sudut kanan atas halaman program
4. Klik butang **Semakan** 
5. Klik butang **Timpa Keputusan** 
6. Kunci masuk **Keterangan** dan buat pilihan **Mohon Pindaan**
7. Klik butang **Kemaskini**
8. Klik butang **Teruskan** 
9. Pastikan status semakan telah dikemaskini kepada status **Perlu pindaan (Oren)**

Outcome:
* Penyedia dan Pengurus program akan menerima notifikasi untuk melakukan pindaan
